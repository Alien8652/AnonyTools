# AnonyTools
Ferramentas para Anonimato

## [Windows]
### Anonymous Cleaner
Limpa uma parte do histórico do Windows.
![alt text](https://github.com/Alien8652/AnonyTools/blob/main/Imagens/Anonymous%20Cleaner.png?raw=true)

### HistoryView

### AnonyNet
